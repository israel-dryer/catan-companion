Overview

This is a companion app intended to be used with the Catan board game, including several variants. The primary purpose of this app is to roll the dice and track the results. The app includes several helpful features such as charts, reports, historical records, sound effects, and more.

Initial Setup

Before you can play a game, you must add players. There are 3 different ways to add players in this app:

  1. There is an onboarding screen that will prompt you to add players when installing the app for the first time.
  2. During game setup you can add additional players by clicking the NEW PLAYER button.
  3. Navigate to Settings and then click Manage Players to add new players.

After creating at least 2 players you are ready to start a new game.

Create a new game
  1. Click the PLAY button on the home screen.
  2. Add at least 2 players by selecting from a list of available players.
  3. Adjust any other game setting as needed such a turn order or toggling an expansion pack.
  4. Click the START GAME button to begin.

Game play
  1. A player is selected to start placing settlements.
  2. Click the START button when all settlements have been placed.
  3. The player to roll is indicated at the top-left by NEXT
  4. The result of the roll is displayed on the screen.

In game options

During the game you may perform any of the following tasks:

  • Click the REPORT button to show a histogram of rolls and also review the roll history.
  • Click the button to perform one or more of the following tasks:
    Undo last roll
    Reverse the previous roll from game history
    End game
    Select a winner (optional) and close the game.
    Settings
    Configure app settings.

Expansion packs

Some expansions are available to play with this app. Any expansion that does not require special dice can be played with this app. Additionally, other select expansions (adding more soon) are available to play as well.

Cities and Knights
  • An additional action dice is rolled along with the standard red and gold dice
  • A appears at the upper-right corner of the screen to indicate how many times the barbarian ship has been rolled. The barbarians attack on roll 7.
  • A button is at the bottom of the screen and is for the Alchemist card. You may adjust the value of the red and gold dice, and then roll the action dice separately.

History

The history page contains a list of all games played. Click on a game to see the details. On the game detail page, you can review the game details as well as perform the following actions:

DELETE
Delete the game from history
CONTINUE
Continue playing a game that was incomplete (a winner was not selected)
REPORTS
Review the reports for this game (histogram and roll history)

Settings

The settings page allows you to control several application level settings such as sound effects, haptics, and voice. You may also perform several other maintenance tasks on this page such as bulk purging game history, resetting app settings, and exporting or sharing the game data.
